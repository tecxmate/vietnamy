### Nouns Grammar Rules

In general nouns refer to a person, an object, or abstract ideas. For example: **a fast runner** the noun is [*runner*] because it refers to a person. The examples below use nouns in different ways and places to demonstrate how they behave in a sentence.

| Grammar + Rules - Vietnamese |
| --- |
| **do you have milk?** *[verb + noun]* bạn có sữa đấy không?  |
| **I have milk and coffee** *[preposition + noun]* tôi có sữa và cà phê  |
| **he has three apples** *[number + plural noun]* ông ta có ba quả táo  |
| **she only has one apple** *[number + singular noun]* cô ta chỉ có một quả táo  |
| **we live in a small house** *[adjective + noun]* chúng tôi sống trong một ngôi nhà nhỏ  |
| **I like our breakfast** *[pronoun + noun]* tôi thích bữa sáng của chúng ta  |

### Conversation in Vietnamese

Now we finally reach the fun part, the practice of the daily conversations. These phrases are used to get to know new people, and break the ice.

| English - Vietnamese |
| --- |
| **Do you speak (English/ Vietnamese)?** bạn có nói (tiếng anh / tiếng việt) không?  |
| **Just a little** chút chút thôi  |
| **I like Vietnamese** tôi thích tiếng việt  |
| **Can I practice with you?** tôi có thể tập cùng với bạn không?  |
| **How old are you?** bạn bao nhiêu tuổi?  |
| **I'm thirty three years old** tôi ba mươi ba (33) tuổi  |
| **It was nice talking to you** rất hân hạnh được nói chuyện với bạn  |
