# Vietnamese Phrases

Welcome to our seventh lesson about popular **Vietnamese phrases**. This page will include **greetings**, **questions**, **emergency and survival expressions**, **asking for direction**, **language practice**, **introducing yourself**, **holiday wishes**, and finally some **travel phrases**.

| Holiday Wishes - Vietnamese |
| --- |
| **Happy birthday** sinh nhật vui vẽ  |
| **Happy new year** chúc mừng năm mới  |
| **Merry Christmas** giáng sinh vui vẽ  |
| **Good luck** chúc bạn may mắn  |
| **Congratulations** chúc mừng  |

  

| Travel Phrases - Vietnamese |
| --- |
| **I have a reservation (hotel)** tôi có đặt phòng trước  |
| **Do you have rooms available?** anh có phòng nào trống không?  |
| **I would like a non-smoking room** tôi muốn phòng không hút thuốc  |
| **What is the charge per night?** tiền mỗi đêm là bao nhiêu?  |
| **Is this seat taken?** chổ này có ai ngồi không?  |
| **I'm vegetarian** tôi ăn chay  |
| **Waiter** bồi bàn  |
| **How much is this?** cái này bao nhiêu tiền?  |
| **This is very expensive** cái này mắc quá  |
