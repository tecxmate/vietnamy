### Numbers Grammar Rules

Vietnamese cardinal numbers refer to the counting numbers, because they show quantity. For example: **I speak two languages**. Ordinal numbers on the other hand tell the order of things and their rank: **my first language is Vietnamese**. The examples below use numbers in different ways and places to demonstrate how they behave in a sentence.

| Grammar + Rules - Vietnamese |
| --- |
| **I have three dogs** *[number + noun]* tôi có ba con chó  |
| **my daughter has two cats** *[number + noun]* con gái tôi có hai (2) con mèo  |
| **she speaks seven languages** *[verb + number]* cô ta nói được bảy ngôn ngữ  |
| **my brother has one son** *[number + singular noun]* anh tôi có một con trai  |
| **this is my second lesson** *[ordinal number + noun]* đây là bài học thứ hai của tôi  |
| **did you read the third book?** *[ordinal number + noun]* bạn đã đọc cuốn sách thứ ba chưa?  |

### Conversation in Vietnamese

Now we finally reach the last part, the practice of the daily conversations. These phrases are used to get to know new people, and break the ice.

| English - Vietnamese |
| --- |
| **Where are you from?** quê quán của bạn ở đâu?  |
| **I'm from the U.S** tôi đến từ mỹ quốc  |
| **I'm American** tôi là người mỹ  |
| **Where do you live?** bạn sống ở đâu?  |
| **I live in the U.S** tôi sống ở mỹ quốc  |
| **What do you do for a living?** anh sinh sống bằng nghề gì?  |
| **I'm a student** tôi là một sinh viên  |
